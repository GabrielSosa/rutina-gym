# 🏋️ Cuerpo Definido — Rutina de Gym 3 Meses

App web de una rutina de gimnasio para principiantes: plan de definición de 12 semanas,
5 días por semana, con técnica paso a paso de cada ejercicio, ajustes de máquina y
videos tutoriales en español.

## Stack

- [React 18](https://react.dev/) + [Vite 6](https://vite.dev/)
- Estilos inline (sin dependencias de CSS)

## Desarrollo

```bash
npm install
npm run dev      # servidor local en http://localhost:5173
```

## Build de producción

```bash
npm run build    # genera /dist
npm run preview  # sirve el build localmente
```

## Deploy

Desplegado en Vercel (autodetecta Vite: build `npm run build`, output `dist`).
